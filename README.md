# MusRS
MusRS is simple command line music player with only 4 command's \

## Usage
```bash
cargo build --release
```
Into release folder you can find executable file `music_player`. Run it with `--path` passing the path to audio file.

List of command's
  - `Play` \
    For playing paused track
  - `Pause` \
    For pause track
  - `Stop` \
    For exit from CLI
  - `Clear` \
    Help you clear command window
